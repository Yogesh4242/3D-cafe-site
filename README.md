# 🎬 Cinematic Scroll Hero + Interactive Menu

A smooth, high-performance scroll experience combining a **frame-by-frame canvas animation** with an **interactive café menu UI**.

---

## ✨ Features

- 🎥 Scroll-synced animation (270 frames)
- 🔄 Sub-frame blending for ultra-smooth playback
- ⚡ Optimized loading with ImageBitmap + lazy loading
- 🧭 Lenis-powered smooth scrolling
- 🎨 Parallax, gradients, and ambient effects
- 🍽️ Interactive horizontal menu with navigation
- 📱 Responsive design

---

## 🛠️ Tech Stack

- React (Next.js)
- TypeScript
- Canvas API
- Lenis (smooth scroll)

---

## 🚀 How It Works

- Scroll position maps to animation frames
- Frames are preloaded and blended for smooth transitions
- Menu overlay fades in towards the end of the scroll
- Users can explore items via horizontal scroll or keyboard
